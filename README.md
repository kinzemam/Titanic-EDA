# Titanic Survival Prediction (EDA + Machine Learning)

This project analyzes the Titanic dataset using:
- Exploratory Data Analysis (EDA)
- Logistic Regression for survival prediction

## Files in this Repository

- `01_Titanic_EDA.ipynb` → Data cleaning & visualization
- `02_Titanic_ML_Model.ipynb` → Model training & evaluation
- `dataset/titanic.csv` → Dataset

## Steps in the Project
1. Data loading and cleaning
2. Exploratory Data Analysis
3. Feature encoding
4. Train-test split
5. Logistic Regression training
6. Model evaluation using accuracy & confusion matrix

## Results
- Model Accuracy: ~80%
- Gender and Passenger Class are the strongest predictors of survival
